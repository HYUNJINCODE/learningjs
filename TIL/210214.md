## C

```c
#include <stdio.h>

int main(void) {
    char a = 65;
    printf("%c\n", a);
}
```

캐릭터형은 자체에 숫자를 넣어서 처리할 수 있다. 문자 입출력에서 형식 지정자로 %c를 쓴다.
한개의 문자 입력 받을 때 getchar() 를 쓴다.

버퍼란 임시적으로 특정한 데이터를 저장하기 위한 목적으로 사용된다.
c프로그램은 기본적으로 사용자가 의도하지 않아도 자동으로 버퍼를 이용해 입출력을 처리한다.
